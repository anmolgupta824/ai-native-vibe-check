# Contributing

## Bug Reports
Open an issue with: what you ran, what you expected, what happened.

## Adding Audiences
New audience JSON files in `config/audiences/` are welcome. Follow the schema in `config/audiences/README.md`. Include platform, segments with scroll_past_triggers, and scoring_weights.

## Pull Requests
- Keep PRs focused — one thing per PR
- Test with `vibe-check battle --dry-run` before submitting
- Don't modify `simulate.py` or `providers.py` without discussion

## Questions
Open an issue or start a Discussion.
